Git is a system used by

Maybe you've emailed a
.docx back-and-forth a few

Some Vocabulary
Git watches
and records
changes to
files.
All the files live in one folder on your computer,
just like any other project you're working on.
Git calls this main project folder a  
Repository
You work with Git by
taking periodic snapshots
of your repository, or Repo.
Git collects and records 
these snapshots, called
commits
Using Git, you push
your own changes to 
the repo and pull the
changes from others.
